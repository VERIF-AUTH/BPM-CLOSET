Publishing settings

Connect the panel to your GitHub repository so Publish updates the shop in one click. You only do this once per device.
Stored only in this browser. Never put it inside the website files.
How to create the token (2 min):
1 · In GitHub: your avatar → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
2 · Name it (e.g. “bpm publish”), set a long expiration, and under Repository access pick Only select repositories → your repo.
3 · Under Permissions → Repository permissions set Contents to Read and write. Generate and copy it here.
